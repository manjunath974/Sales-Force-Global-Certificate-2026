About The Project

Building smarter, greener, and more efficient cities

🌆 Project Overview

The Smart Adaptive Street Light System is an AI-assisted IoT dashboard designed to automate street light scheduling for urban environments. It intelligently manages lighting patterns based on the day of the week and automatically adapts to holidays by shifting schedules to the next valid working day.

🎯 Key Objectives

  • Reduce unnecessary electricity consumption
  • Automate lighting decisions without manual intervention
  • Adapt intelligently to holidays and special days
  • Provide real-time monitoring through a clean dashboard
  • Support smart city infrastructure initiatives
